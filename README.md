🐾 Mouse's Enchanted Adventures
Welcome to the official source code for Mouse’s Enchanted Adventures, an immersive, interactive, and enchanted children’s website built by Katherine McNeil.

This website features:

A whimsical portal into Mouse’s magical universe
An interactive AI companion called Chat Katherine: The Digital Familiar
Products, a magical café entrance, and a space for readers to connect
🌐 Live URLs
Main Website: Hosted on Netlify/GitHub Pages (custom domain, if connected)
Live Chat (Render): https://mouse-chatbox.onrender.com/
Digital Familiar Page: /digital-familiar.html
✨ Features
Fully mobile responsive
Themed navigation bar with animated hamburger menu
Glowing interactive button to summon Chat Katherine
Magical gradients, custom illustrations, and whimsical font choices
Pages styled consistently with bright, playful energy
🧠 The Digital Familiar (AI Assistant)
Custom-built landing page: digital-familiar.html
Backend API hosted on Render: https://mouse-chatbox.onrender.com/
Connected via JavaScript fetch() requests
Fully themed to match site aesthetic
Live chatbot using OpenRouter or HuggingFace for AI replies
