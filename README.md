- **Backend API**: Hosted on [Render.com](https://render.com)

- Uses modern `fetch()` calls in JavaScript

## 🚀 How to Deploy

### 🌐 Frontend
Deploy all HTML, CSS, and JavaScript using:
- [Netlify](https://netlify.com)
- [GitHub Pages](https://pages.github.com)
- Or your **custom domain**

### 🧠 Backend (AI Chat API)
1. Sign up and deploy via [Render](https://render.com)
2. Use environment variables to store API keys (like `OPENROUTER_KEY`)
3. Host the `/chat` endpoint to respond to AI prompts
